3 types of css

Inline,Internal and External

All the Types are explained below in brief

Inline css involves using style attribute within the element .This is not commonly used as its not a good practice,its used when we need to style a particular element or to make quick changes

Internal css involves using style tag inside the head element,its used for bulk edit and suitable for styling single web page

External css uses .css file to place all the styles in that single file.This file can be used for multiple pages after writing it once.
